# Learnings Memory — Aprendizajes

Guardar aquí lecciones generales que cambian cómo trabajamos.

Ejemplos:

- Un reporte de agente no es evidencia.
- Una doc de cliente se valida contra código real.
- Un deploy no está listo sin smoke browser.
- Una función mobile no está lista sin device real cuando toca nativo.

Cada aprendizaje debe convertirse en una regla práctica.
