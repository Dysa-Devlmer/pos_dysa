# Ideas Memory — Ideas y Mejoras Futuras

Guardar aquí ideas antes de convertirlas en roadmap.

Cada idea debe indicar:

- Problema que resuelve.
- Usuario beneficiado: dueño, cajero, Dyon Labs, soporte, agente.
- Valor comercial.
- Complejidad estimada.
- Dependencias.
- Riesgos.
- Recomendación: implementar, investigar, diferir o descartar.

Las ideas no son promesas. Solo se vuelven promesa cuando están
implementadas, verificadas y documentadas como funcionalidad real.
