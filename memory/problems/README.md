# Problems Memory — Problemas, Bugs y Riesgos

Guardar aquí problemas vivos o resueltos que no deben repetirse:

- Bugs críticos.
- Brechas entre manual y código.
- Riesgos de seguridad.
- Riesgos de datos.
- Deuda técnica.
- Deuda operativa.
- Promesas no implementadas.

Cada problema debe incluir síntoma, causa real, impacto, evidencia,
solución y estado actual.
