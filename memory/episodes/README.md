# Episodic Memory — Recuerdos de Sesiones y Eventos

Guardar aquí recuerdos concretos: sesiones, deploys, APK builds,
validaciones, auditorías, incidentes y smoke tests reales.

Cada nota debe responder:

- Qué ocurrió.
- Cuándo ocurrió.
- Quién lo ejecutó.
- Qué evidencia existe.
- Qué cambió después de ese evento.
- Qué quedó pendiente.

Formato de nombre:

```text
YYYY-MM-DD-<tema>.md
```
