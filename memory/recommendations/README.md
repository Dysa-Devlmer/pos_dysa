# Recommendations Memory — Recomendaciones Vigentes

Guardar aquí recomendaciones técnicas, comerciales u operativas que
siguen vigentes.

Cada recomendación debe incluir:

- Qué se recomienda.
- Por qué.
- Riesgo si se posterga.
- Quién debe decidir.
- Fecha o condición de revisión.
- Estado: activa, aceptada, rechazada, implementada o reemplazada.
