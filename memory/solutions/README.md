# Solutions Memory — Soluciones y Patrones Probados

Guardar aquí soluciones reutilizables:

- Fixes que resolvieron errores reales.
- Comandos exactos.
- Runbooks cortos.
- Patrones técnicos aceptados.
- Antipatrones que se deben evitar.

Una buena solución debe explicar cuándo aplica, cuándo no aplica y qué
verificación demuestra que funciona.
